# Memmath

Dodawanie liczb sposobem pisemnym.

## Założenia projektowe

- Funkcja przyjmuje conajmniej dwa parametry.
- Liczby powinny być przechowywane jako tekst.

## Przykład użycia

### C

```bash
$ cd c
$ make
$ ./cli 456 789

   111
    456
+   789
-------
   1245
```

### PHP

```bash
$ cd php
$ ./cli.php 456 789

  111
   456
+  789
------
  1245
```
